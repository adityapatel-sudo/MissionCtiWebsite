Intermediate Python

6 classes, 1.5 hours each

Class Timings

  1. Thursday 1:15 - 2:45
  2. We will increase class counts with sign-ups

Objectives

Our second Python course is for students who want to learn more complex programming topics, and to continue to improve at Python. This class will cover using complex IDEs such as Pycharm and programming topics of Object Oriented Programming, Inheritance, Recursion, and more. Students in Intermediate Python are expected to have a grasp of all concepts from our Introductory Python class.

This class is open to all students grade 5 and up.

Details

  1. Review of variables, loops, operators, lists, functions, and practice problems. Installation of Pycharm and explanation of the IDE.
  2. Error checking, intro to OOP theory, and more problems to become comfortable with previous materials
  3. Further in depth with classes, OOP, and inheritance in Python.
  4. Recursion, practice problems with Recursion
  5. explanation and intro to algorithms, Review and practice
  6. Complete Session Review, practice problems with all learned in class
