Objectives

Our second Java course is for students who want to learn more complex programming topics, and to continue to improve at Java. This class will cover topics of Object Oriented Programming, Inheritance, Recursion, and more. Students in Intermediate Java are expected to have a grasp of all concepts from our Introductory Java class.

This class covers many of topics that students encounter in AP Computer Science A, and is a great way to prepare for the class and the AP Exam.

This class is open to students grade 5 and up.

Details

  1. Intro to OOP
  2. Writing Classes and inheritance
  3. Practice with iteration, Arrays, Arraylist, and 2D Arrays
  4. Continued practice problems and brief intro to recursion
  5. Code Logic Problems and Code Testing
  6. Session Review and Practice
