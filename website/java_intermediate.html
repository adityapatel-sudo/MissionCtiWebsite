Intermediate Java

6 classes, 1.5 hours each

Class Timings

  1. Mondays 10:30 AM

Objectives

Our second Java course is for students who want to learn more complex programming topics, and to continue to improve at Java. This class will cover topics of Object Oriented Programming, Inheritance, Recursion, and more. Students in Intermediate Java are expected to have a grasp of all concepts from our Introductory Java class.

Details

  1. Introduction to Python, Repl.it setup, Python Syntax, Basic Functions, Math and Boolean operators
  2. Variables, Casting, Operations
  3. If & Else Conditions, Comparators
  4. Lists, Indexing & Slicing
  5. For Loops, While Loops, Review
  6. Session Review and Practice
  1. A working computer, laptop or tablet
  2. Zoom, with a working microphone and webcam
  3. Google Classroom
  4. Book and Pencil, or other notetaking device
  1. Enter Zoom Meeting at Class Start time
    1. Make sure to arrive on time!
    2. Have materials and HW ready to discuss
  2. First 15-20 minutes
    1. Instructors and Students review HW questions and answers
    2. Previous lessons recap
  3. Lesson | 45 Minutes
    1. Interactive Lessons
    2. Introduction to new concepts
    3. Multiple problems completed with students
  4. Activity | Final 30 Minutes
    1. Review Lesson
    2. Trivia, Practice Problems, ect.
    3. Assign and explain HW problems & dismissal
